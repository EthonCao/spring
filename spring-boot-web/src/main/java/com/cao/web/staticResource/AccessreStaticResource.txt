spring boot 有2种访问访问静态资源的方法
 
 1， spring boot 从classpath/static目录下访问静态资源，目录名称必须时“static”
 	本工程的访问示例为：
 	http://localhost:8080/index.html
 	http://localhost:8080/images/java.jpg
 
 2，从ServletContext目录下
 	maven工程中的ServletContext目录在src/main下， 即：src/main/webapp， 目录名称必须为webapp
 	本工程的访问示例为：
 	http://localhost:8080/index2.html
 	http://localhost:8080/images2/java2.jpg
